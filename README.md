# HBCodeSnippets

##快捷键
<br>
1、PS ,对应：

```
@property (nonatomic, strong) NSString *<#name#>;
```

2、PI,对应：

```
@property(nonatomic, assign) NSInteger <#name#>;
```

3、PB,对应：

```
@property (nonatomic, assign) BOOL <#name#>;
```

4、WS，对应：

```
__weak typeof(self) weakSelf = self;
```

5，SS，对应：

```
__strong typeof(weakSelf) strongSelf = weakSelf;
```
